# 🎯 AI YouTube Thumbnail Generator

Generate eye-catching YouTube thumbnails automatically using AI, computer vision, and video analysis.

This project takes a YouTube video URL, extracts key visual moments, uses AI to create a compelling thumbnail headline, and generates a ready-to-use thumbnail image in seconds.

---

## 🚀 Features

* Download videos directly from YouTube
* Extract important video frames automatically
* Generate AI-powered thumbnail text
* Create professional thumbnail layouts
* Add overlays and visual enhancements
* Export ready-to-upload YouTube thumbnails
* Fully automated workflow

---

## 📸 How It Works

```text
YouTube URL
    ↓
Download Video
    ↓
Extract Frames
    ↓
Analyze Video Content
    ↓
Generate Thumbnail Headline (AI)
    ↓
Create Thumbnail Design
    ↓
Export Final Thumbnail
```

## 📋 Requirements

Before running the project, users need:

### 1. OpenAI API Key

Create an API key from OpenAI and add it to the notebook.

```python
OPENAI_API_KEY = "YOUR_API_KEY"
```

### 2. YouTube Video URL

Provide any public YouTube video URL.

Example:

```text
https://youtube.com/watch?v=XXXXXXXXXXX
```
